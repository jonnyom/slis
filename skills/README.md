# Skills

This directory contains Claude Code skills that teach Claude how to drive `slis`. Copy `skills/slis/` into `~/.claude/skills/slis/` (for personal use across all projects) or into a project's `.claude/skills/slis/` (to scope it to that repo) so that Claude automatically knows the right `slis` commands for listing slices, creating feature worktrees, activating/deactivating slices, generating summaries, and setting up hooks.
